# 🧠 Dynamic Programming — Java

A collection of classic Dynamic Programming problems implemented in Java using **Recursion**, **Memoisation**, and **Tabulation** approaches.
This repository contains my practice programs on **Recursion and Dynamic Programming (DP)** implemented in Java.
These programs helped me understand different DP techniques and commonly asked interview problems.

## 📁 Files
## Programs

* **Recursion.java** – Basic program to understand recursion.
* **DpMemoisation.java** – Fibonacci using Dynamic Programming with memoization.
* **DpTabulation.java** – Fibonacci using Dynamic Programming with tabulation method.
* **ClimbingStairs.java** – Find the number of ways to climb stairs.
* **FrogJump.java** – Minimum energy required for a frog to reach the last step.
* **HouseRobber1.java** – Maximum money that can be robbed without robbing adjacent houses.
* **HouseRobber2.java** – Circular version of the House Robber problem.
* **MaxValueNonAdjacent.java** – Find maximum sum of non-adjacent elements.
* **EqualPartition.java** – Check if an array can be divided into two subsets with equal sum.
* **Knapsack01.java** – Implementation of the 0/1 Knapsack problem.
* **SubsetSum.java** – Determine if a subset with a given sum exists.
* **UnboundedKnapsack.java** – Knapsack problem where items can be chosen multiple times.
* **EditDistance.java** – Minimum operations required to convert one string into another.
* **LongestCommonSubsequence.java** – Find the longest common subsequence between two strings.
* **LongestPalindromicSubsequence.java** – Find the longest palindromic subsequence in a string.
* **ShortestCommonSupersequence.java** – Find the shortest string containing two given strings as subsequences.
* **FallingPathSum.java** – Find the minimum falling path sum in a matrix.
* **LongestIncreasingSubsequence.java** – Find the longest increasing subsequence in an array.
* **MinPathSum.java** – Find the minimum path sum in a grid.
